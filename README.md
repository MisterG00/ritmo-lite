Ritmo Lite

Juego rítmico minimalista desarrollado con HTML5 Canvas, JavaScript puro
y Web Audio API.

Descripción

Ritmo Lite es un mini–juego rítmico donde el jugador presiona SPACE
cuando el círculo dinámico coincide con el círculo objetivo.

Incluye: - Sistema de precisión (PERFECT / MISS) - Score y Combo -
Efectos visuales - Sonido de golpe - Música de fondo - Diseño
minimalista - JavaScript puro

Controles

-   Golpear: SPACE
-   Iniciar música: Primer SPACE o clic

Estructura del Proyecto

audio/ hit.wav music.mp3 game.js index.html README.md

Cómo Ejecutarlo

1.  Live Server en VS Code
2.  O abrir index.html directamente

Tecnologías Utilizadas

HTML5 Canvas, JavaScript ES6, Web Audio API, requestAnimationFrame

Lógica del Juego

1.  Loop de juego: anima y dibuja
2.  Precisión: compara radios
3.  Puntaje: PERFECT +100, MISS reinicia combo
4.  Audio: hitSound + música de fondo
